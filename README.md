# max_projects

This repository contains all my MaxMSP 9 projects, organized into folders for easy navigation.

## How to Use the Projects

1. Clone this repository to your computer.
2. Open MaxMSP 9.
3. Open the `.maxpat` file contained in the project folder you want to use.
